Projeto final da disciplina de Introdução à Ciência da Computação.

Para execução do algorítimo é necessário:
1. Ter instalado em máquina o Python versão 3.13 ou superior (estável);
2. Ter instalado as seguintes bibliotecas: Numpy, Pandas, Matplotlib e Seaborn (versões compatíveis com a do Python).

Instrução de execução:
- Com o terminal de seu computador, na pasta do arquivo 'main.py', execute o comando 'python3 main.py'.

  (pode variar um pouco dependendo da versão do Python instalada, mas em essência é executar o arquivo 'main.py' com o python)
